# what do you need

 * [Node.js] [1]
 * [Npm] [2] ([video tutorial (fr)] [4])
 * [Yarn][5]


# Install Yarn

    sudo npm install --global yarn

# How to install the project

  * If it is the first installation
    * `yarn install`
  * if it's a re installation
    * `rm -rf node_modules`
    * `yarn install`

# Build the project

#### prod
   > yarn start
