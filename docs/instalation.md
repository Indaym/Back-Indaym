# what do you need

 * [node.js] [1]
 * [npm] [2] ([video tutorial (fr)] [4])


# How to install the project

You have one thing to do:

    > npm install

    if this command is too long to write in the terminal, this is the equivalent:

      npm i

# Build the project

#### prod
   > npm start
